mEAL (minimal EAL) is a version of EAL, a MOD validation tool used by the phpBB modifications team designed for use by modification writers. mEAL creates HTML reports about the entire structure of a MOD prior to submission to the phpBB modifications database. mEAL performs the following checks:

  • phpBB MOD Template validation

  • MODX validation

  • PHP function usage validation (to MOD Team requirements)

  • HTML usage validation (to phpBB requirements)

  • MODX packaging requirements

System Requirements

mEAL is designed run on many systems and has been tested under the following platforms with the following recommended hardware

Linux

  • Any Linux distribution running MONO; which can be downloaded from http://www.mono-project.com/Downloads

  • 32MB RAM

  • 200MHz processor, faster recommended

  • 5MB HDD space

MAC OSX

  • MONO; which can be downloaded from http://www.mono-project.com/Downloads

  • 32MB RAM

  • 200MHz processor, faster recommended

  • 5MB HDD space

Windows

  • Windows 98, NT4, 2000, XP or Vista

  • The .net framework 1.1 or MONO 1.1

  • 32MB RAM

  • 200MHz processor, faster recommended

  • 5MB HDD space

Using mEAL

Linux, MAC OSX and MONO on Windows®

  1. Make sure you have the minimum system requirements

  2. If you haven’t already, download mEAL from phpBB.com

  3. Extract mEAL.zip if you haven’t already

  4. Start a terminal window/bash/command window

  5. Type the following (using the path meal.exe is located, here it is /root/release/)

    [root@fedora ~]# mono /root/release/meal.exe

  6. The following will appear

pp      pp  hh          pp      pp  BB      BB  BB      BB

pppppppp    hhhhhhhh    pppppppp    BBBBBBBB    BBBBBBBB
pp          hh      hh  pp          BB      BB  BB      BB
pp          hh      hh  pp          BB      BB  BB      BB
pp          hh      hh  pp          BBBBBBBB    BBBBBBBB
----------------------------------------------------------
The phpBB MOD Team MOD pre-validation script.
Please enter the path to a folder of zip to validate:

  7. Type in the name of the folder of zip file to validate, for example

    /root/release/sql_backup/sql_backup_v1.0.2/

  8. Press Enter.

  9. The file will be validated with a number of status messages filling the screen, feel free to ignore these as they are repeated in the HTML report

  10. All done, a file named report.htm will have been created and should have opened in your default browser.

  11. Examine the HTML report generated for anything you need to fix. Not all reported issues may be relevant or indeed be issues. Use commonsense to dictate which issues are indeed issues, and need to be fixed.

The .net Framework on Windows® (best for windows)

  1. Make sure you have the minimum system requirements

  2. If you haven’t already, download mEAL from phpBB.com

  3. Extract mEAL.zip if you haven’t already

  4. You now have several options

    1. Drag the folder of zip file you wish to validate onto the executable file (meal.exe) and it will be validated automatically. A file named report.htm will have been created and should have opened in your default browser.

    2. Open a command prompt by going, start » run » type "cmd" » Enter.

      1. Type the following (using the path meal.exe is located, here it is c:\mEAL\)

        C:\>c:\mEAL\meal.exe
